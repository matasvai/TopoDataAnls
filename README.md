# TopoDataAnls
Topological Data Analysis method that are applicable to Explainable AI interventions to classify types of topological shapes and patterns that appear. 
Originally devolped on PEEK to observe shapes from entropy maps to compare topological similarity, further devolped to be more applicable. PEEK: 
## M. Meni, T. Mahendrakar, O. D. Raney, R. T. White, M. L. Mayo, and K. R. Pilkiewicz (2024). Taking a PEEK into YOLOv5 for Satellite Component Recognition via Entropy-based Visual Explanations. AIAA SCITECH 2024 Forum. https://arc.aiaa.org/doi/abs/10.2514/6.2024-2766
